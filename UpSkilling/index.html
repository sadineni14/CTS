Exercise 1 — Inline vs Internal vs External CSS

This page uses all three CSS methods:

  • Inline: The <h1> above has style="color:#ffeb3b"
  • Internal: The <style> block in <head> sets body background
  • External: styles.css is linked via <link> and contains all reusable rules

Exercise 10 — Responsive Web Design

Resize your browser window or use Chrome DevTools Device Toolbar to test responsiveness. Key breakpoints:

  • ≤ 992px (Tablet): max-width becomes 100%, header font shrinks
  • ≤ 768px (Mobile): nav stacks vertically, images & fonts shrink, 1-column layout
  • ≤ 480px (Small mobile): buttons go full-width, tighter padding

Exercise 11 — Debug & Test with DevTools

To debug and test this page:

  1. Open Chrome DevTools with F12 or right-click → Inspect
  2. Use the Device Toolbar (Ctrl+Shift+M) to simulate phones & tablets
  3. In the Elements panel, select any element and edit styles live in the Styles pane
  4. In the Network tab, reload the page and confirm styles.css loads with status 200
  5. In the Console tab, view any JS logs or CSS warnings
